## 📖 Description

Forkify is a recipe search and saving app, built with **HTML5**, **CSS3**, and **JavaScript**. It features an intuitive user interface, dynamic serving size adjustment, and advanced JavaScript techniques such as asynchronous programming, ES6 modules, and object-oriented programming.

Originally created by Jonas Schmedtmann in his Udemy course, this version was built as a personal project for learning and practice.

---

## ✨ Features

- 🔍 **Search for Recipes**: Find recipes using keywords or ingredients.
- 📖 **View Recipe Details**: Get ingredients, instructions, and serving size.
- 📌 **Bookmark Recipes**: Save favorite recipes for later use.
- 📝 **Add Recipes**: Create and share your own recipes.
- 🔢 **Pagination**: Browse through search results easily.
- 📱 **Responsive Design**: Looks great on all screen sizes.
